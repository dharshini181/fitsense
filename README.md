# ✨ Outfit Recommender App

**Description:**  
Outfit Recommender App is a Streamlit web app that suggests personalized outfits based on gender, mood, occasion, subtype, and weather. It recommends dresses or top-bottom combos with color palettes, prints, footwear, accessories, and buy links for a complete style guide.

## Features

- Personalized outfit recommendations based on:
  - Gender
  - Mood
  - Occasion
  - Subtype (e.g., Dress, Saree, Top/Bottom)
  - Weather
- Displays:
  - Dress or Top & Bottom
  - Color palette & print types
  - Footwear and accessories
  - Buy link
- Friendly UI with success and warning messages
- Modular and easy to extend
